# Часть 3: Использование готовых MCP-серверов

В **части 1** мы изучили основы tool calling - как модели вызывают инструменты через structured output.

В **части 2** мы создали собственный MCP-сервер и научились запускать его через STDIO и HTTP транспорты.

В **этой части** мы научимся использовать **готовые MCP-серверы**.

## Концепция готовых MCP-серверов

Многие сервисы и компании предоставляют готовые MCP-серверы как NPM/Python пакеты. Вместо написания собственного сервера для каждого API, вы просто устанавливаете пакет и используете его.

### Преимущества

- Не нужно писать серверный код
- Готовая интеграция с популярными сервисами
- Автоматические обновления
- Проверенная реализация

### Репозитории с готовыми MCP-тулами:

- https://hub.docker.com/mcp/explore
- https://glama.ai/mcp/servers
